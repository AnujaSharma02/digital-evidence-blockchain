CSE 435 - Comprehensive Seminar Demo

Register Evidence

Upload one or more files (or a whole folder). The system zips multiple files into a single deterministic archive, generates a unique Case ID, and records its SHA-256 fingerprint.

Verify Evidence

Upload the original file(s) and enter the Case ID. If you registered multiple files, select the same files again — they will be re-zipped identically for comparison.

Tamper Demo

Upload any file (or multiple files) to corrupt it. Download the corrupted version, then submit it to Verify to see tamper detection in action.
